## Correctly



## installation 
```bash
git clone https://github.com/Khush-S/Correctly
cd Correctly

# Create a virtual environment
python3 -m .venv venv

# Activate the virtual environment
# On Windows
venv\Scripts\activate
# On macOS/Linux
source venv/bin/activate

#install dependencies
pip install -r requirements.txt



## run
python app.py

 
`
